## Tumhari CNC ka actual program style

Real programs me pattern ye hai:

* program start: `O5000`
* header comment with file/date
* tool comment: `( T02 )`, `( T04 )`, `( T08 )`
* startup:

  * `G0`
  * `G53`
  * `G28 U0.`
  * `G28 W0.`
  * `M1`
* tool call:

  * `N2 / N4 / N8`
  * `T0202 / T0404 / T0808`
* spindle limit:

  * `G92 S500`
* cutting mode:

  * `G96 S200 M4`
* absolute mode:

  * `G90`
* safe start:

  * `G0 Z50.`
* cutting:

  * repeated `G0` approach + `G1` contour passes
  * arcs with `G3 ... R0.8`
* end style should stay consistent with your post and machine-home logic

Ye generic lathe output nahi hai.
Ye **tumhari machine/operator habit ke hisaab ka real proven format** hai.
