# ⚙️ 2. ROLL CONTOUR ENGINE (MOST CRITICAL)

👉 Ye hi decide karega ki drawing production usable hai ya nahi

## Roll contour generate karne ke rules:

### A. Basic relation

```
Roll contour = final profile - forming stage offset
```

### B. Compensation factors

| Factor      | Effect          |
| ----------- | --------------- |
| Thickness   | gap define      |
| Material    | springback      |
| Bend radius | corner shape    |
| Pass no.    | gradual forming |
| Return bend | extra clearance |

---

## C. Springback Compensation

Approx rule:

| Material | Compensation |
| -------- | ------------ |
| GI       | 1–2°         |
| MS       | 2–3°         |
| SS       | 3–5°         |

👉 App me add karo:

```text
target_angle = required_angle + compensation
```

---

## D. Roll Gap Calculation

```
gap = thickness + clearance
```

| Thickness | Clearance |
| --------- | --------- |
| <1 mm     | +0.05     |
| 1–2 mm    | +0.1      |
| >2 mm     | +0.15     |
