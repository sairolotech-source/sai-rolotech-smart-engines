# Dimension aur angle kyu nahi aa rahe

Iske common reasons:

**1. Bend detection incomplete**

* app ko 1 bend mil gaya
* but uska exact angle calculate/display nahi kiya

**2. Auto annotation layer missing**

* profile ready hai
* but screen par labels draw karne wala engine run nahi ho raha

**3. Center line conversion output me metadata missing**
Converted profile me ye fields absent ho sakti hain:

* dimensions[]
* angles[]
* bendAnnotations[]
* overallWidth
* flange lengths

**4. UI only status dikha rahi hai, detailed profile summary nahi**
Ho sakta hai backend me value ho, par frontend par show hi nahi ho rahi.
