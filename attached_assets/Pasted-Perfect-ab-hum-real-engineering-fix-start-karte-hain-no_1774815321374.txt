# 🔴 STEP 1 — FLAT STRIP WIDTH (MOST CRITICAL FIX)

![Image](https://static.thefabricator.com/a/analyzing-the-k-factor-in-sheet-metal-bending-1578853830.jpg)

![Image](https://static.thefabricator.com/a/have-problems-calculating-the-flat-in-sheet-metal-bending-1636481721.jpg)

![Image](https://static.thefabricator.com/a/what-sheet-metal-shops-wish-you-knew-minimum-flange-bend-radius-and-tooling-access-1499885439.jpg)

![Image](https://www.solitaire-overseas.com/images/channel-weight.png)

## ✅ Correct Formula

```
Flat Length = Web + 2×Flange + Σ(Bend Allowance)
```

## Bend Allowance (BA):

```
BA = (π/180) × Angle × (R + K × T)
```

### Where:

* Angle = bend angle (°)
* R = inner radius
* T = thickness
* K = K-factor (GI ≈ 0.42–0.45)

---

## 🔧 Example (aapka case)

* Web = 60
* Flange = 40 + 40
* T = 1.5
* R = 1.5
* K = 0.44
* Angle = 90° × 2 bends

```
BA per bend = 1.57 × (1.5 + 0.44×1.5)
= 1.57 × (1.5 + 0.66)
= 1.57 × 2.16
= 3.39 mm

Total BA = 6.78 mm

Flat = 60 + 80 + 6.78 = 146.78 mm
```

👉 Engine ka 65 mm → **completely wrong**
