# SAI Rolotech Smart Engines v2.2.0

  ## Download
  This repo contains the full source code of SAI Rolotech Smart Engines.

  ### Quick Download
  Download the ZIP from this repo using the green "Code" button → "Download ZIP"

  ### Setup (Windows)
  1. Extract the ZIP
  2. Run `setup-laptop.bat`
  3. App starts automatically

  ### Features
  - Roll Forming Design
  - CNC G-Code Generator  
  - DXF Import/Export
  - AI Diagnostics
  - 3D Visualization
