# ⚙️ 1. DEFORMATION ENGINE (CORE)

👉 har pass par profile transform kare

```python
def simulate_pass(profile, ratio):
    new_profile = []
    for x, y in profile:
        new_profile.append((x, y * ratio))
    return new_profile
```

---

# ⚙️ 2. STRAIN CALCULATION

```python
def calculate_strain(radius, thickness):
    return thickness / (2 * radius)
```

---

# ⚙️ 3. SPRINGBACK MODEL

```python
def springback(material, angle):
    factor = {
        "GI": 1.5,
        "MS": 2.5,
        "SS": 4.0
    }.get(material, 2)

    return angle + factor
```

---

# ⚙️ 4. FORCE ESTIMATION

```python
def forming_force(thickness, width, strength):
    return thickness * width * strength * 0.8
```

---

# ⚙️ 5. DEFECT ENGINE

```python
def detect_defects(strain, ratio, thickness):
    issues = []

    if strain > 0.3:
        issues.append("Cracking risk")

    if ratio > 0.9 and thickness < 1:
        issues.append("Wrinkling risk")

    return issues
```

---

# ⚙️ 6. FULL SIMULATION LOOP

```python
def run_simulation(profile, passes, thickness, material):
    results = []

    for i in range(passes):
        ratio = (i + 1) / passes

        new_profile = simulate_pass(profile, ratio)

        strain = calculate_strain(5, thickness)
        angle = 90 * ratio
        corrected = springback(material, angle)

        defects = detect_defects(strain, ratio, thickness)

        results.append({
            "pass": i + 1,
            "profile": new_profile,
            "strain": strain,
            "angle": corrected,
            "defects": defects
        })

    return results
```
