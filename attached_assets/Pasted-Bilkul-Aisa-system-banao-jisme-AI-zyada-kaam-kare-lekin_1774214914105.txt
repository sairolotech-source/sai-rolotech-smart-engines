## Real system ka setup

### 1) Machine ko standard karo

Har machine ke liye ye lock karo:

* controller type
* VMID / machine definition
* correct `.gpp` postprocessor
* machine simulation file
* max spindle/feed limits

SolidCAM docs ke mutabik machine details me postprocessor aur machine simulation dono machine definition se linked hote hain. ([SolidCAM Help][3])

### 2) Tool library standard karo

Example:

* T1 Face mill
* T2 12 mm end mill
* T3 6 mm end mill
* T4 center drill
* T5 drill
* T6 chamfer tool

Rule:

* same machine par same tool number standard rakho
* holder aur stickout bhi fixed rakho

Isse AI ka output repeatable ho jayega.

### 3) SolidCAM templates banao

Har common kaam ke liye ready templates:

* facing
* pocket roughing
* contour roughing
* contour finishing
* drilling
* chamfer

Template me pehle se:

* clearance plane
* safety plane
* stepdown
* stepover
* lead in/out
* coolant logic
* default feed strategy

### 4) AI output format fixed karo

AI ko free text me nahi, fixed format me bolo.

Use this prompt:

```text
You are a CNC process planner.

Machine: VMC
Controller: Fanuc
Material: EN8 / MS / Aluminum
Available tools:
T1 Face Mill 50
T2 End Mill 12
T3 End Mill 6
T4 Drill 6
T5 Chamfer 45 deg

Give output in this exact format:

SETUP_1:
- Workholding:
- WCS:
- Stock:
- Risks:

OPERATIONS:
1. Facing | Tool | Purpose | Safe depth notes
2. Rough Pocket | Tool | Stepdown | Stepover
3. Finish Contour | Tool | Finish allowance
4. Drill | Tool | Peck yes/no
5. Chamfer | Tool | Notes

CUTTING_NOTES:
- RPM range:
- Feed range:
- Coolant:
- Entry style:
- Thin wall warning:

FINAL_CHECK:
- Clamp collision risk:
- Tool reach risk:
- Retract risk:
- Datum check:
```

## SolidCAM me actual sequence

1. CAM-Part banao
2. Machine select karo
3. Sahi postprocessor `.gpp` choose karo
4. Toolpath calculate karo — SolidCAM docs ke hisab se G-code se pehle calculation zaroori hai. ([SolidCAM Help][4])
5. Simulate karo — official simulation operation mistakes aur unsuitable strategy detect karne me help karti hai. ([SolidCAM Help][5])
6. Machine Simulation chalao
7. G-code generate/display karo — output controller aur postprocessor par depend karta hai. ([SolidCAM Help][2])

## Zero-risk ke 7 locks

1. **Verified postprocessor**
2. **Correct VMID / machine definition**
3. **Tool + holder exact**
4. **Clamp model included in simulation**
5. **High safe retract**
6. **Dry run first**
7. **First-piece inspection**
