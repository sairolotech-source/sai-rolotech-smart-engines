SECTION 2 — APP MODULE ARCHITECTURE
==================================================

Split the app into the following modules:

A. IMPORT ENGINE
- import DXF / SVG / JSON / supported drawing formats
- detect entities: line, arc, polyline
- detect unit system
- detect scale mismatch
- verify profile completeness
- detect open vs closed geometry
- normalize coordinate data

B. GEOMETRY ENGINE
- extract profile edges
- calculate dimensions
- detect bends
- detect flange lengths
- detect web depth
- detect radii if available
- calculate overall width and height
- convert centerline to profile if required
- validate section geometry

C. INPUT ENGINE
- ask for sheet thickness
- ask for raw material
- validate thickness range
- validate supported materials
- store confirmed input in structured form

D. FLOWER PATTERN ENGINE
- segment bends into passes
- distribute forming sequence
- calculate progressive bend development
- generate stage-wise forming logic
- detect simple / medium / complex profile category

E. STATION ESTIMATION ENGINE
- estimate required no. of forming stations
- use complexity rules
- use thickness/material corrections
- use bend count and section depth
- return min and recommended station count

F. ROLL DESIGN LOGIC ENGINE
- generate preliminary roll forming logic
- estimate pass breakdown
- estimate roll groups / forming progression
- assign rough roll sequence
- log all assumptions

G. MECHANICAL SELECTION ENGINE
- estimate shaft diameter
- suggest shaft thickness / strength class
- suggest bearing type based on shaft size and load category
- assign light / medium / heavy duty classification
- note safety factors and assumptions

H. REPORT ENGINE
- produce structured JSON
- produce readable engineering summary
- show warnings
- show missing-data alerts
- allow export later
