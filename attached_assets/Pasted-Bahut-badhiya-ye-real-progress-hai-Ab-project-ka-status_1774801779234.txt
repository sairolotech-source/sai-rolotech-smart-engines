if (!station.rollProfile) status = "No Profile";

### 2) **T007 — Fake Success Removal**

Ye trust ke liye mandatory hai.

**Abhi rule ye hona chahiye:**
success tabhi show ho jab:

* code path complete hua ho
* UI preview render hua ho
* export object bana ho
* validation pass ho gayi ho

**Matlab:**
sirf function chal gaya = success nahi
sirf toast dikha diya = success nahi

**Recommended gate:**

```ts
const canMarkSuccess =
  geometryValid &&
  profileGenerated &&
  flowerGenerated &&
  stations.every(s => s.rollProfile) &&
  exportPayloadReady;
```

Agar ye fail ho:

* green success mat dikhao
* exact failure message dikhao

Example:

* `DXF parsed but flower generation failed`
* `9 me se 3 stations incomplete`
* `Export blocked: rollProfile missing`

### 3) **T010 — Null Guards / Crash-Proof UI**

Ye React stability ke liye abhi bhi जरूरी hai.

Tumne pehle bhi dekha tha:
`rollProfile undefined` wali wajah se crash aa raha tha.

**Ab har jagah safe render chahiye:**

* tables
* exports
* station detail panel
* digital twin
* BOM
* roll tooling views
* CSV/XML generators

**Pattern:**

```ts
if (!station?.rollProfile) {
  return <EmptyState message="No roll profile generated" />;
}
```

Aur direct access kabhi nahi:

```ts
station.rollProfile.rollDiameter
```

Isko safe banao:

```ts
station?.rollProfile?.rollDiameter ?? "-"
```
