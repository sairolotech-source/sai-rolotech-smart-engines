# CORE UPGRADE FEATURES

---

## FEATURE 1 — MULTI VIEW MODES (VERY IMPORTANT)

Split drawing into 3 modes:

### MODE 1: BASIC (Operator Mode)

* show only:

  * profile
  * station number
  * bend angle
  * strip path
* minimal dimensions
* clean and simple
* large readable visuals

### MODE 2: ENGINEERING (Designer Mode)

* full dimensions
* gap, depth, OD, bore
* section view
* roll contours
* annotations

### MODE 3: MANUFACTURING

* full roll drawing
* tolerances
* material
* keyway
* shaft
* title block
* revision

### RULE:

User must be able to switch modes instantly without rerunning pipeline.

---

## FEATURE 2 — ANIMATED FLOWER PATTERN

Add a button:

👉 **“Play Flower Pattern”**

### Behavior:

* animate stations sequentially:
  Station 1 → 2 → 3 → ... → final
* current station = bold color
* previous stations = faded grey
* final profile = red dashed

### Show:

* % completion
* current bend angle
* movement direction

### Controls:

* play / pause
* speed control
* manual slider

---

## FEATURE 3 — PROFILE → FLOWER → ROLL LINKED FLOW

Create a visual pipeline:

👉 Profile → Developed Strip → Flower Pattern → Station Geometry → Roll Shape

### RULE:

All stages must be linked.

When user selects station:

* flower pattern updates
* roll geometry updates
* drawing updates

User should understand:
👉 “ye roll shape kis step se bani hai”

---

## FEATURE 4 — CONTACT POINT VISUALIZATION

Highlight where strip touches rolls.

### Show:

* contact points = red dots
* pressure zones = shaded area
* non-contact zones = light outline

### Must:

* dynamically update per station
* reflect gap and depth

---

## FEATURE 5 — AUTO EXPLANATION SYSTEM

For each station generate explanation:

Example:

* “Station 1: edge pre-forming”
* “Station 2: flange lifting”
* “Station 3: web stabilization”
* “Station 4: near final shaping”
* “Station 5: calibration pass”

### Also explain:

* what changed from previous station
* why this station is needed

---

## FEATURE 6 — STATION LOGIC ENGINE (ADVANCED)

Add button:
👉 **“Explain This Station”**

System must output:

* purpose of station
* bend progression logic
* risk if removed
* merge possibility (e.g. 12 → 8 stations)

---

## FEATURE 7 — COLOR STANDARDIZATION

Use fixed colors:

* strip = gold
* upper roll = blue
* lower roll = green
* current stage = orange
* final profile = red dashed
* centerline = grey dashed
* dimensions = black

## FEATURE 8 — 3D VISUAL PREVIEW (LIGHTWEIGHT)

Add simple 3D preview:

* upper roll
* lower roll
* strip passing through
* rotate / zoom

## FEATURE 9 — DESIGN WARNING ENGINE

Detect issues:

### Show warnings:

* roll clash
* excessive depth in one pass
* too tight radius
* unrealistic gap
* strip pinching
* impossible machining geometry

### Behavior:

* show warning icon
* show reason
* suggest fix if possible

* center line

* outside profile

### Modes:

* center offset
* inside fixed
* outside fixed
