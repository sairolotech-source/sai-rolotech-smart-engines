## C. Center Line to Sheet Conversion Engine

Ye core engine hai.

### Rules

Agar imported drawing center line hai:

* profile ko neutral axis maan ke
* thickness/2 dono side offset

Agar inner face profile hai:

* outward full thickness offset

Agar outer face profile hai:

* inward full thickness offset

### Auto detection logic

Software try kare detect karne:

* centerline
* inner profile
* outer profile

Agar confidence low ho:

* operator prompt de

### Thickness handling

* minimum thickness
* nominal thickness
* maximum thickness
* tolerance band

Example:

* min: 0.75
* nominal: 0.80
* max: 0.85

Ye bahut important hai kyunki flower pattern aur roll gap dono thickness band par depend karenge.
