# Client Demo Storyboard

**App:** SAI Rolotech Smart Engines v2.3.0
**Target Duration:** ~2 minutes

## Scene 0: Logo reveal + tagline (0:00–0:12)
- **Visuals:** Video background of roll forming. SAI Rolotech logo and "Smart Engines v2.3.0" appear. "Near COPRA-class Roll Forming Engineering Platform" tagline. Counters for 720 automated tests, 29 engine modules, 45 API endpoints.
- **Voiceover:** "SAI Rolotech Smart Engines — a near COPRA-class roll-forming engineering platform. Designed for precision. Built for production."

## Scene 1: Material + Profile Input (0:12–0:30)
- **Visuals:** Blueprint texture overlay. Material card showing GI DX51D, K=500MPa, n=0.22, Fy=250MPa. Profile sketch of a lipped channel with T=2mm, W=120mm. Animated parameters filling in.
- **Voiceover:** "Start by selecting your material — Galvanised Steel DX51D — and defining the section profile. A lipped channel, 2 millimetre thick, 120 millimetres wide. Load a saved project or enter parameters directly."

## Scene 2: Engineering Core (0:30–1:05)
- **Visuals:** 13 stations showing angle progression. Flower diagram with 13 nested profile outlines showing progression. 
- **Voiceover:** "The station engine calculates 13 forming passes, distributing bend angle incrementally across each station. Watch the flower progression unfold — each pass bringing the flat strip closer to the final profile. Roll contour is generated automatically — upper and lower rolls matched to the strip geometry. Springback correction applied per pass. Force, power and torque calculated at each station."

## Scene 3: Manufacturing Outputs (1:05–1:35)
- **Visuals:** Close-up of steel roll tooling background. Manufacturing cards highlighting Bill of Materials, Process Card, Tooling Library, and One-Click CAD Export (DXF, STEP, CAD pack).
- **Voiceover:** "Bend allowance and flat blank width — calculated instantly. Bill of Materials generated with shaft, bearing and roll specifications. Process card ready for the shop floor. Tooling library saves your roll sets for reuse. Export to DXF, STEP, or a full CAD pack — one click."

## Scene 4: Simulation + Proof (1:35–1:52)
- **Visuals:** "Advanced Process Simulation Precheck" title. Pass-by-pass stress, strain, contact analysis. Risk indicators: PASS (Stress), WARNING (Wrinkling), PASS (Edge Wave). FEA-ready solver decks (CalculiX/Abaqus) indicated.
- **Voiceover:** "Advanced Process Simulation Precheck — pass-by-pass stress, strain and contact analysis. Defect probability, fracture risk, wrinkling and edge wave flags — all instant. FEA-ready solver decks generated for CalculiX and Abaqus when you need full simulation."

## Scene 5: Close (1:52–2:00)
- **Visuals:** Metal strip through rollers background. Large logo and summary: "Engineering-grade precision. From input to export." Large stats: 720 tests, 29 modules, 45 endpoints.
- **Voiceover:** "720 automated tests. 29 engine modules. 45 live API endpoints. SAI Rolotech Smart Engines — engineering-grade precision, from input to export."
