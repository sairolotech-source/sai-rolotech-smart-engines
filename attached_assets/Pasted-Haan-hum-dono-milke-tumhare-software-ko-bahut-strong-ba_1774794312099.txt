## “Without error” ke liye real strategy

Ye 8 layers lagani hongi:

### 1. Guard layer

* `?.`
* `if (!rp) return`
* safe defaults

### 2. Repair layer

* stale localStorage repair
* missing rollProfile synthesis
* missing gcode regeneration

### 3. Validation layer

* `VALID`
* `BASIC`
* `MISSING_PROFILE`
* `INVALID_GEOMETRY`

### 4. Export gating

* VALID = full export
* BASIC = geometry only
* MISSING = blocked

### 5. Test layer

* unit tests
* scenario tests
* migration tests
* export tests

### 6. Debug layer

* per-station debug summary
* exact diagnostics
* warn on skips

### 7. Mechanical audit layer

* groove width
* wallRun
* pass line
* complementarity
* interference

### 8. CNC safety layer

* safe retract
* spindle stop
* home reference
* no empty programs

## Ab tumhe kya karna chahiye

Ye **single master instruction** Codex ko do:

You are not allowed to optimize for speed or cosmetic cleanup.
Optimize for correctness, validation, and non-crashing behavior.

Primary goal:
Make the roll forming software truthfully reliable.

Rules:

1. Never mark incomplete engineering output as complete.
2. Never allow empty geometry arrays to pass as valid.
3. Never allow empty G-code strings to pass as valid.
4. Never rewrite working modules without a proven defect.
5. Always audit first, fix second, test third.
6. Every fix must include:

   * exact failing line
   * failing condition
   * minimal diff
   * rerun result
7. Every station must end in one of:

   * VALID
   * BASIC
   * MISSING_PROFILE
   * INVALID_GEOMETRY
8. BASIC and MISSING states must never crash the app.
9. All exports must reflect true capability.
10. Report only genuine engineering issues.

Execution order:
A. Audit
B. Isolate failures
C. Minimal fix
D. Run tests
E. Print final proof report
