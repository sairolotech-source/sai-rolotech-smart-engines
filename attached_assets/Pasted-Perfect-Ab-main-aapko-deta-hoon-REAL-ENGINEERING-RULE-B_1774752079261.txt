## ⚙️ 1. PROFILE COMPLEXITY CLASSIFICATION

![Image](https://www.researchgate.net/publication/288767364/figure/fig5/AS%3A868748915396608%401584137290178/Channel-section-assemblies.png)

![Image](https://tubecon.co.za/img/Wiki/coldformedlipchannel.PNG)

![Image](https://roll-formingmachines.com/products/2-7-yx33-56-hat-profile-roll-forming-machine_01.jpg)

### SIMPLE

* 2–4 bends
* No lips / no return bends
* Example: U channel

### MEDIUM

* 4–8 bends
* Lips present
* Partial symmetry

### COMPLEX

* 8–12 bends
* Return bends
* Narrow flange / deep section

### VERY COMPLEX

* 12+ bends
* Shutter / multi-return / zig-zag
* Tight geometry

👉 **Rule**

```text
complexity = f(bend_count + return_bends + lip + depth_ratio)
```

---

## 📐 2. BEND COUNT DETECTION RULE

```text
bend_count = number_of_angle_changes in profile
```

### Important:

* 90° = 1 bend
* Return bend = 2 bends
* Lip = +1 bend
* Arc = treat as 1 bend

👉 Agar bend_count detect nahi hua → flower pattern FAIL

## 🌸 3. FLOWER PATTERN BASIC LOGIC

![Image](https://www.researchgate.net/publication/332100955/figure/fig3/AS%3A742282202460162%401553985276873/Roll-forming-split-flower-diagram-13.ppm)

![Image](https://www.researchgate.net/publication/226012190/figure/fig12/AS%3A668556303806466%401536407652036/Typical-bending-types-applied-to-progressive-die-design.png)

![Image](https://www.researchgate.net/publication/334097046/figure/fig8/AS%3A779423494205441%401562840450631/Illustration-of-the-roll-forming-machine-a-the-working-principle-of-the-torque.png)

![Image](https://www.iqsdirectory.com/articles/roll-forming/rolling-forming-process.jpg)

### Golden Rule:

👉 **Ek hi pass me full bend mat karo**

### Distribution logic:

```text
total_passes ≈ bend_count + correction
```

### Example:

| Profile        | Bend | Pass  |
| -------------- | ---- | ----- |
| Simple channel | 2    | 4–6   |
| Lipped channel | 4    | 6–10  |
| Complex        | 8    | 10–16 |

### Standard progression:

1. Edge pickup
2. Pre-form
3. Intermediate bend
4. Lip forming
5. Closing
6. Calibration

---

## 🏭 4. STATION COUNT RULE MATRIX

```text
stations = base + complexity_factor + thickness_factor + material_factor
```

### BASE RULE:

* 1 bend ≈ 1–1.5 station

### Correction factors:

#### Complexity:

* SIMPLE → +0
* MEDIUM → +2
* COMPLEX → +4
* VERY COMPLEX → +6

#### Thickness:

* <0.8 → +0
* 0.8–1.2 → +1
* 1.2–2 → +2
* > 2 → +3

#### Material:

* GI → +0
* MS → +1
* SS → +2

👉 Final:

```text
recommended_station = bend_count + all_corrections
```

![Image](https://ars.els-cdn.com/content/image/3-s2.0-B9780080977591000071-f07-01-9780080977591.jpg)

### Basic classification:

| Duty       | Shaft  |
| ---------- | ------ |
| Light      | 40 mm  |
| Medium     | 50 mm  |
| Heavy      | 60 mm  |
| Very Heavy | 70+ mm |

### Decision logic:

```text
if thickness < 0.8 AND simple → light
if thickness 0.8–1.2 → medium
if thickness >1.2 OR complex → heavy
if shutter / high load → very heavy
```

## 🏗️ 7. DUTY CLASS (MACHINE STRENGTH)

```text
duty = f(thickness + complexity + material)
```

| Condition     | Class      |
| ------------- | ---------- |
| Thin + simple | LIGHT      |
| Medium        | MEDIUM     |
| Thick/complex | HEAVY      |
| Shutter       | INDUSTRIAL |
