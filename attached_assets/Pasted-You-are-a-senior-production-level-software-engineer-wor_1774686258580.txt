3. IMPACT ANALYSIS

4. MINIMAL FIX PLAN

5. CODE CHANGE

6. VERIFICATION

   * Step-by-step test

7. RISK CHECK

   * What might break

8. CONFIDENCE LEVEL

   * % (if <90% → warn)
